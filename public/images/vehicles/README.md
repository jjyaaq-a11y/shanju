# 车辆选项图片（本地）

把 4 张车辆图片放到此目录，文件名与 `src/lib/addons.ts` 中 `vehicle[].previewImage` 对应即可。

建议文件名（可自定，在 addons.ts 里改路径即可）：

- `standard-business-van.jpg` — 标准商务车（如塞纳）
- `luxury-business-van.jpg` — 豪华商务车
- `standard-suv.jpg` — 标准 SUV（如 Tank 300）
- `luxury-suv.jpg` — 豪华 SUV

支持 .jpg / .jpeg / .png / .webp。
